# Agentic Graph - (Beta)
A open source AI agent builder made with **Next.js**. Built on top of LangChain and LangGraph, it's mainly targeted towards devolepers.
This project is still in development and being developed.

Status:
![CiCd](https://github.com/StickyCoolDev/AgenticGraph/actions/workflows/format.yaml/badge.svg)


## Project Stack
1. **Next.js 16**
2. **Shadcn UI and Tailwind**
3. **Prettier and ESlint**
4. **Firebase** 
5. **IronSession**
6. **TypeScript**
